常见问题

我的数据保存在哪里？

所有数据仅保存在你的设备本地，不会上传到云端。建议定期使用「导出完整备份」保存可恢复的 JSON 备份；Excel 适合查看和整理，不用于恢复。

如何开启 Face ID 解锁？

进入「我的」→「隐私安全」，打开「面容 ID 解锁」开关即可。打开后每次进入 App 需要验证身份。

如何导出礼簿数据？

进入「我的」→「数据与备份」→「导出 Excel」，即可生成 .xlsx 文件并分享或保存。

如何导入 Excel？

进入「我的」→「数据与备份」→「导入 Excel」，选择 .xlsx 文件后先查看预览。有效记录会新增到礼簿，重复项会自动跳过，不会覆盖现有数据。

如何合并重复记录？

进入「我的」→「数据与备份」→「去重合并」。App 只会列出姓名、收送类型、金额、事件和日期均一致的明确重复项，确认后保留信息更完整的一笔。

如何设置送礼或回礼提醒？

入簿时展开「更多信息」，开启送礼或回礼提醒并选择日期和时间；再到「我的」开启「系统通知」。通知仅在 iPhone 本地安排。

如何备份和恢复全部数据？

进入「我的」→「数据与备份」，选择「导出完整备份」。需要恢复时选择「从备份恢复」，确认内容数量后再替换当前礼簿。

忘记 Face ID 验证怎么办？

Face ID 验证失败后，系统会自动提供设备密码作为备用验证方式。输入设备密码即可解锁。

如何删除数据？

在礼簿中左滑单条记录可删除，或在记录详情页操作。卸载 App 将删除所有本地数据。
